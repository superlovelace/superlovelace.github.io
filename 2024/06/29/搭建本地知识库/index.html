准备

FastGPT 官方文档 ：https://doc.fastgpt.in/docs/development/docker/

系统环境：Windows系统 + Docker Desktop

主要参考官方文档中 --> 使用 Docker Compose 快速部署 FastGPT

开始

首先创建文件目录，以fastgpt文件夹名为例。然后进入该目录，在文件目录输入cmd，进入终端。
